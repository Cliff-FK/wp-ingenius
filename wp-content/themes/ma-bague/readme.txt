== Description ==

Le thème Ma-Bague est un thème hybride conçu pour exploiter une partie de la puissance de l’éditeur de blocs, tout en orientant l’utilisateur vers des options simples et minimalistes. Il comporte des styles personnalisés pour tous les blocs proposé, et est construit pour que ce que vous voyez dans l’éditeur ressemble à ce que vous verrez sur votre site Web. Ce thème est conçu pour s’adapter à un large éventail de sites Web, que vous exploitiez un blog photo, que vous lanciez une nouvelle entreprise ou souteniez un organisme à but non lucratif. Doté d’un large espace blanc, de titres modernes, couplés à un corps de texte lisible, il est conçu pour être beau sur toutes les tailles d’écran.
